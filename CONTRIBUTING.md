# Contributing

Welcome! We appreciate you checking out the project and we are excited to see what you can contribute.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone the project** to your own machine.
3.  **Commit changes** to your own branch.
4.  **Push your work** back up to your fork.
5.  Submit a **Pull Request** so that we can review your changes.

## Development

-   Make sure to create a virtual environment (`python -m venv venv`).
-   Install dependencies via `pip install -r requirements.txt`.
-   Follow standard Python PEP 8 coding style.

## Code of Conduct

Please note that this project is released with a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
